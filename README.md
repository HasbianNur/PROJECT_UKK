# Ternak Lelang

Ternak Lelang adalah Website Lelang Berbasis Laravel 9

## Installation

Pertama Jalankan Composer Update

```bash
composer update
```

## Usage

```python
import foobar

# returns 'words'
foobar.pluralize('word')

# returns 'geese'
foobar.pluralize('goose')

# returns 'phenomenon'
foobar.singularize('phenomena')
```
